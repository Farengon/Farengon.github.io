# AP Computer Science A 教程 💻

欢迎来到 AP CSA 核心知识点梳理。本教程完全基于 **College Board 2025-2026 年最新考纲**设计，帮助你掌握 Java 编程核心。

## 📚 课程结构（新考纲 4 单元）

| 单元 | 名称 | 考试权重 | 课时 |
|------|------|----------|------|
| **Unit 1** | Using Objects and Methods | 15-25% | 32-34 |
| **Unit 2** | Selection and Iteration | 25-35% | 29-31 |
| **Unit 3** | Class Creation | 10-18% | 20-22 |
| **Unit 4** | Data Collections | 30-40% | 50-52 |

## 📖 学习建议

1. 按顺序学习各个单元
2. 每个 Topic 都包含基础知识、代码示例和练习题
3. 多动手写代码，实践出真知！
