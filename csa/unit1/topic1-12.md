# 1.12 Objects: Instances of Classes

## 基础知识

### 类和对象
- **类 (Class)**: 对象的蓝图或模板
- **对象 (Object)**: 类的具体实例

### 类比
- 类 = 建筑图纸
- 对象 = 根据图纸建造的具体房子

## 代码示例

### 示例 1: String 对象
```java
public class StringObjects {
    public static void main(String[] args) {
        String word1 = "Java";
        String word2 = "Programming";
        
        System.out.println(word1);
        System.out.println(word2);
    }
}
```

## 例题

### 例题 1: 选择题
以下哪个说法是正确的？
A. 类是对象的实例
B. 对象是类的实例
C. 类和对象是同一个概念
D. 一个类只能有一个对象

**答案: B**
