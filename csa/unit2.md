# Unit 2：使用对象 (Using Objects)

本单元我们开始接触面向对象编程（OOP）的概念。

## 1. 什么是类和对象？

* **类 (Class)**: 对象的蓝图或模板（例如 `String` 类，或者自定义的 `Student` 类）。
* **对象 (Object)**: 类的具体实例。

## 2. 实例化一个对象

使用 `new` 关键字来调用构造函数：
```java
String greeting = new String("Hello World");
// 或者简写
String greeting2 = "Hello World";
```
