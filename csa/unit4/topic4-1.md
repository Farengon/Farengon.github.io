# 4.1 Ethical and Social Issues Around Data Collection

## 基础知识

### 数据隐私
- 个人数据应该受到保护
- 用户应该知道他们的数据如何被使用

### 数据安全
- 防止数据泄露
- 加密敏感信息

## 代码示例

### 示例 1: 概念性
```java
// 注意保护用户隐私
public class DataPrivacy {
    public static void main(String[] args) {
        System.out.println("Always respect user privacy!");
    }
}
```

## 例题

### 例题 1: 选择题
以下哪项是重要的数据伦理原则？
A. 收集所有可能的数据
B. 保护用户隐私
C. 公开所有数据
D. 忽略用户同意

**答案: B**
