# 4.2 Introduction to Using Data Sets

## 基础知识

### 数据集
数据集是数据的集合，用于分析和处理。

## 代码示例

### 示例 1: 简单数据集
```java
public class DataSet {
    public static void main(String[] args) {
        int[] scores = {85, 92, 78, 90, 88};
        
        for (int score : scores) {
            System.out.println(score);
        }
    }
}
```

## 例题

### 例题 1: 选择题
数据的集合称为？
A. 数组
